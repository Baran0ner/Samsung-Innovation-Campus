# Lab 2 Image Generation Prompts

## 1. Who I Am

Create a clean, professional, cinematic widescreen personal branding visual
for a Computer Engineering student. Show a young student with glasses working
on a laptop in a modern dark-blue technology workspace. Surround the student
with subtle interface overlays representing artificial intelligence, machine
learning, data analytics, cloud computing, databases, and software systems.
Include visual cues for problem solving, innovation, attention to detail, and
continuous learning. Use realistic lighting, crisp details, cyan-blue accents,
and a polished student portfolio aesthetic. Avoid logos and watermarks.

## 2. My Skills

Create a modern widescreen technology portfolio visual titled around turning
data into intelligent solutions. Present a refined dark-blue workspace with
clear interface panels for Python code, SQL databases, AI and machine learning,
data analysis dashboards, automation workflows, cloud storage, computing, and
security. Include a laptop on the desk and use a clean professional layout,
realistic lighting, cyan highlights, sharp readable visual hierarchy, and a
high-end data engineering dashboard aesthetic. Avoid logos and watermarks.

## 3. My Future Goal

Create a cinematic widescreen visual showing the future goal of building
AI-powered systems that scale. Show a Computer Engineering student designing
an end-to-end intelligent platform with connected panels for an AI assistant,
data sources, ETL and ingestion, a data lake, AI/ML models, model operations,
automation workflows, APIs and integrations, cloud infrastructure, monitoring,
and business dashboards. Emphasize practical enterprise use cases, reliability,
automation, and real business impact. Use a professional dark navy environment,
glowing cyan connections, realistic depth, and a polished technology portfolio
style. Avoid logos and watermarks.
